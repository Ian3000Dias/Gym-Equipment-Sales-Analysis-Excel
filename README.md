# Gym Equipment Sales Analysis | Excel Data Analysis

## Project Overview

This project analyzes profitability trends across multiple fitness equipment brands, suppliers, and product categories using Microsoft Excel.

The objective was to transform raw transactional data into structured business reports that highlight performance trends, profit contributions, and year-over-year changes. Using Pivot Tables and Excel reporting techniques, the analysis provides a clear view of business performance across several years.

This project demonstrates core data analysis skills frequently used in business intelligence, reporting, and data analyst roles.

---

## Business Problem

Organizations need to understand which products, brands, and suppliers contribute most to profitability in order to make informed business decisions.

This analysis focuses on answering questions such as:

* Which brands generate the highest profits?
* How has profitability changed over time?
* Which suppliers contribute most to overall performance?
* What percentage of total profit is driven by each category and brand?
* Which products show the strongest year-over-year growth?
